Although it is possible to program in a text editor, it can be quite a pain. Trying to use the command line for the first time could have also been daunting in the last lesson. Good news, this isn't how most programmers develop software. An easier way to write, compile and run code is within an integrated development environment (IDE). An IDE contains all the necessary tools to program without having to open up a command line interface. For these lessons I will be using Code::Blocks. I would suggest downloading this because it is the easiest way to get started on programming in the c language. Code::Blocks is open source and works on virtually any operating system. Download it here.

Once you have downloaded it, open it in your downloads folder and follow the instructions to install. I suggest leaving everything to its default settings. Once it is open it should look very similar to this:

Close the Start here tab. Click File then hover over New. Select the Empty File option. Type in the code we made last time. If you don't remember I have provided it below: 

	#include <stdio.h>
	
	int main()
	{
		printf("Hello World");
		return(0);
	}
Now click File then Save File As and choose where you want your file to be saved to and name it. Remember to include the .c extension. You should notice that colors have been added to each element of the program. This will make longer programs easier to read.

At this point, you could click Build then Compile and then Run from the Build tab in the menu, but it is just easier if you hit F9. This will do all three tasks at once. A terminal interface should pop up with the words Hello World displayed. And that's it! We'll talk about how this program works in the next section. Click here to go to the next lesson.
